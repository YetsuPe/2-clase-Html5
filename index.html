Sprites,Fuentes y tipografías font-face

Sprites

Es un método que consiste en utilizar una sola imagen que a su vez agrupa muchas más y que haciendo uso de las propiedades background-position de CSS se visualice cada imagen de forma individual, nos sirve mucho para ahorrar peticiones al servidor y disminuir tamaño a las imágenes

Desventajas :

Al no ser imagenes vectoriales no podemos redimensionarlas sin perder caldad :(

                        
                            .ico{
                                width: 64px;
                                height: 64px;
                            }
                            .buttons_sociales { 
                                background:url(image/redes-sociales.jpg) no-repeat
                            }
                            .rss {background-position: 0 0 }
                            .twitter {background-position: -64px 0 }
                            .digg {background-position: -128px 0 }
                            .stumbleupon {background-position: -192px 0 }
                            .facebook {background-position: 0 -64px }
                            .delicious {background-position: -64px -64px }
                            /*** Otra alternativa ****/
                            .display-relative{position: relative}
                            .rss2{left:0;}

Fuentes y tipografías font-face

Es uno de los nuevos Features que nos proporciona CSS3, es la más recomendada al momento de trabajar con iconos, al ser fuente tipografíca nos proporciona iconos vectoriales lo cual hace mas ligero el manejos de imágenes y podemos redimenzionar sin perder calidad

Desventajas :

No podemos tener iconos coloridos, son colores solidos, ya que cada icono es equivalente a una letra 

                        
                            @font-face {
                              font-family: 'FontAwesome';
                              src: url('../font/fontawesome-webfont.eot?v=3.0.1');
                              src: url('../font/fontawesome-webfont.eot?#iefix&v=3.0.1') format('embedded-opentype'),
                                url('../font/fontawesome-webfont.woff?v=3.0.1') format('woff'),
                                url('../font/fontawesome-webfont.ttf?v=3.0.1') format('truetype');
                              font-weight: normal;
                              font-style: normal;
                            }
                            [class^="icon-"],[class*=" icon-"] {
                              font-family: FontAwesome;
                              font-weight: normal;
                              font-style: normal;
                              text-decoration: inherit;
                              -webkit-font-smoothing: antialiased;
                            }
                            .icon-twitter:before{ content: "\f099"; }
